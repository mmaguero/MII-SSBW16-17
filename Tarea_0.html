Entorno virtual python


Lo primero que haremos preparar nuestro entorono de desarrollo, usaremos un entorno virtual para tener controladas siempre las versiones de software que usemos, de manera que sean las mismas, tanto en desarrollo, como las que se instalen en producción.

Lo podemos hacer como en Setting up Python 3, Django and Gunicorn on Ubuntu 16.04 LTS, y de paso instalamos Django y gunicorn, que usaremos en el desarrollo

No está demás usar git, para control de versiones y como backup, auque sea poniendo el repositorio remoto en github for free o incluso en un pen: Setting up a repository
